# unity-selenium

unity-selenium is a repo for Selenium part of homework 2.

## Setup
1. Clone repo 
`git clone https://gitlab.cs.ttu.ee/htoots/unity-selenium`
2. Make sure you have Maven
3. Run tests through IDE or through command line with: `mvn verify`
